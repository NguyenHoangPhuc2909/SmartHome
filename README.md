# SmartHome

Mục đích của dự án: Hệ thống quản lý nhà thông minh tích hợp nhận diện khuôn mặt).

## Tính năng chính
- Nhận diện khuôn mặt với OpenCV.
- Điều khiển thiết bị qua giao diện Web (React).
- Quản lý dữ liệu cảm biến thời gian thực.

## Công nghệ sử dụng
- **Frontend:** ReactJS, Vite, Tailwind CSS.
- **Backend:** Python (Flask), MySQL.
- **Hardware:** ESP32, ESP32-CAM.

## Hướng dẫn cài đặt
1. Clone dự án: `git clone <link-repo>`
2. Cài đặt Backend: `pip install -r requirements.txt`
3. Cài đặt Frontend: `npm install`
